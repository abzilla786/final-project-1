# Setting up

describe how to setup local environment

### getting the
```
git clone <repo ssh or https>
```

to create your local environment please
```
vagrant up
```

<!-- to see the website with your changes go to
```
development.local
```-->

to run the app:
```
vagrant ssh
cd /home/ubuntu/app

python3 main.py
```

output of app and report go to:
```
/home/vagrant/downloads
```
